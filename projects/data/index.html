营销智能数据助手

2019-10-09 — Design by Charlie

产品介绍 #

营销智能数据助手是为营销中心提供专业数据洞察的智能平台，为解决房企中下游数据监管难题，利用移动端灵活的特性，满足随时随地进行数据监控和分析，准确把控和调整企业营销重点，提高管理者的决策效率。

建立数据指标 #

通过与营销管理中心的业务员讨论分析，了解到管理者在决策过程中所重点关注的指标信息，通过筛选和梳理得到以下关键业务指标。其中还包括对各个指标的详细分析，这里暂不赘述。

选择可视化图表 #

选择什么图表，需要遵循「 有什么数据指标，需要用图表做什么 」的设计原则，根据达成指标分析目的考虑，再将不同图表一一对号入座。按照业务上定义好的指标，所需图表主要以下面几种类型为主。

结合以上图表类型，将前面所收集到的关键业务指标进行归纳，整理出图表对照清单，在设计时作为参照。

结构设计 #

结构按照先总览后细化的方式，界面视图按照不同角色身份和业务考察需要，至上而下，展示不同的数据信息和图表信息。

图表布局 #

图表布局主要包含坐卡片标题、筛选项、图表示例图、图表容器、坐标轴，另外还可以包括辅助标记、数据标签等。

视觉设计 #

可视化设计中，色彩也设计的重要环节之一，如果色彩使用不当也会带来糟糕的体验问题。所以在产品配色上，以品牌主题色为主，基于HSB色彩模型中的色调（H）、饱和度（S）、明度（B）进行取色调配，在图表上使用颜色时，则根据不同数据类型与颜色进行关联。

定义字体 #

为保证字体的识别性和统一性，针对数字字符的使用也进行了分析比较，选用了兼容性更好的Roboto字体，作为数据展示的标准字体。

验证业务 #

经过两个月的设计和开发，智能数据助手在碧桂园凤凰云营销中心上线并使用，通过对营销数据指标的采集-分析-呈现-驱动，形成了持续性的数据监测，达到预期效果，从营销管理层到销售人员，能够在移动场景下，随时随地打开手机查看和分析数据，不再错过解决问题的最佳时间，提高决策效率。同时，该项目部分设计方案作为标准化产品方案进行整合，为后续产品SaaS化提供了设计基础。
